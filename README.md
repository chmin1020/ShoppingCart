#쇼핑리스트 앱

<p>kotlin 기반 개발, 지난 프로젝트(recipeDiary)에서 겹치는 코드 참고</p>
<p>주요 기능 -> 쇼핑리스트 저장 / 수정/ 삭제, 각 아이템 체크 및 해제</p>
<p style="background-color:green;">RecyclerView와 Room Database 활용, 코틀린 코드 자체는 조잡한 면이 있어 추후 보완 예정</p>
